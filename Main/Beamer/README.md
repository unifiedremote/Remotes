# Beamer
Remote control for the Beamer Mac app when streaming video from your Mac to Apple TV.  
http://beamer-app.com/

## Features
* Show player info (state, position, duration, title)
*  Next movie
*  Previous movie
*  Stop playback
*  Toggle play/pause state
*  Set position
*  Open files using built-in file picker

* Mac OSX
