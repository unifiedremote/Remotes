# Firefox
Mozilla Firefox browser remote.

## Features
*  Navigation (back, forward, home)
*  Tabs (open, close, next, previous)
*  Find on current page
*  Refresh current page
*  Scroll up/down
*  Zoom in/out
*  Focus URL field
